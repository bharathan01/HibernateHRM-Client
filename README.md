# HRM System

## Overview
HRM System is a web-based Human Resource Management application built using Angular and Material UI. It provides a user-friendly interface for managing employees, job applications, and other HR-related tasks efficiently.

## Features
- **Employee Management**: Add, update, and delete employee records.
- **Job Applications**: Track and manage applications.
- **Dashboard**: Visual reports and insights.
- **Authentication**: Secure login with JWT authentication.
- **Material UI Components**: Responsive and accessible UI.
- **Lazy Loading**: Optimized performance with Angular modules.
- **Real-time Notifications**: WebSocket integration for instant updates.

## Tech Stack
### Frontend
- **Angular**: Framework for building dynamic web applications.
- **Angular Material**: UI component library for sleek and consistent design.
- **RxJS**: Reactive programming for handling asynchronous data streams.
- **SCSS**: Styling with modular and reusable stylesheets.
- **NgRx (Optional)**: State management for large-scale applications.

## Installation
### Prerequisites
Ensure you have the following installed:
- **Node.js** (Latest LTS version recommended)
- **Angular CLI** (`npm install -g @angular/cli`)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/HRM-System.git
   cd HRM-System
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   ng serve
   ```
4. Open the application in your browser at:
   ```
   http://localhost:4200
   ```
